# 🚀 Nome do Projeto / SauceDemo

[E-commerce fictício](https://sauce-demo.myshopify.com/) chamado sauce demo para validação de fluxos. O objetivo deste projeto é demonstrar a aplicação de boas práticas de Quality Assurance, cobrindo desde o planejamento estratégico até a automação de testes funcionais.


## 🛠️ Ferramentas & Tecnologias

* **Automação de Testes:** Cypress
* **Controle de Versão:** Git & GitHub
* **Linguagem:** JavaScript / TypeScript
* **Gerenciamento de Testes & Bugs:** Jira, Markdown, Github issues.
* **CI/CD:** GitHub Actions
* **Plataforma de API:** Postman
